## Getting Started

First, install dependencies and run the development server (pnpm preferred):

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/pages/HomePage.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Features

- Floating AI chat widget on every page (bottom-right), see `src/app/components/ChatWidget.tsx` wired in `src/app/layout.tsx`.
- App Router structure with presentational pages under `src/app/pages/*`.
- Tailwind CSS v4 and Geist fonts preconfigured.
